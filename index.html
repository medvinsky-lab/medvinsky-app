Receptor-ligand interactions across the human embryonic hematopoietic stem cell niche

Interactively explore receptor-ligand interactions as predicted by NATMI across the human embryonic hematopoietic stem cell niche using spatial (Crosse et al., 2021; Crosse et al., 2020) and single cell (Crosse et al., 2021, Zeng et al., 2019) datasets. Toggle through the different datasets available including three spatial transcriptomes from Carnegie stages 13 (LCM-Seq CS13), 14 (LCM-Seq CS14) and 16 (LCM-Seq CS16), and the hematoendothelial + niche single cell dataset.

Tutorial

  1. Select dataset using the dataset dropdown menu
  2. Select ligand population using the ligand dropdown menu
  3. Select receptor population using the receptor dropdown menu
  4. Toggle between the heatmap and interaction plots using the buttons
  5. Use the heatmap to view overall interaction strengh between populations
  6. The interaction plot shows the top 15 interactions between selected populations

Receptor-ligand interactions across the human embryonic hematopoietic stem cell niche

Interactively explore receptor-ligand interactions as predicted by NATMI (link) across the human embryonic hematopoietic stem cell niche using spatial (Crosse et al., 2021; Crosse et al., 2020) and single cell (Crosse et al., 2021, Zeng et al., 2019) datasets. Toggle through the different datasets available including three spatial transcriptomes from Carnegie stages 13 (LCM-Seq CS13), 14 (LCM-Seq CS14) and 16 (LCM-Seq CS16), and the hematoendothelial + niche single cell dataset.

Tutorial

  1. Select dataset of interest via the dropdown menu
  2. Select a ligand population by clicking on a spatial domain or cluster
  3. Select a receptor population by clicking on a second spatial domain or cluster
  4. Toggle between the heatmap and interactions view using the upper-left buttons
  5. View the overall interaction strength between two populations using the heatmap view
  6. View the top 20 interactions for the selected populations using the ligand-receptor view
